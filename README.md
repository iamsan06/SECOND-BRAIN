# 🧠 AI Second Brain (v0.1)

An experimental AI-powered Second Brain built using FastAPI, Obsidian notes, and LLM APIs.  
This project aims to create a personal knowledge system that can store, search, and understand notes using AI.

---

## 🚀 Features

- Chat with your notes
- Obsidian notes integration
- FastAPI backend
- LLM API support (Groq)
- Basic RAG (Retrieval Augmented Generation)
- Simple frontend UI
- Environment-based API key handling

---

## 🧱 Tech Stack

- Python
- FastAPI
- JavaScript / HTML / CSS
- Obsidian (Markdown notes)
- LLM API (Groq)
- dotenv

## ⚙️ Setup

Clone repo:

```
git clone https://github.com/your-username/ai-second-brain.git
cd ai-second-brain
```

Install dependencies:

```
pip install -r requirements.txt
```

Create `.env` file:

```
GROQ_API_KEY=your_key_here
```

Run backend:

```
uvicorn main:app --reload
```

Open frontend manually in browser.
